Saya mengikuti Bootcamp Laravel Web Development karena ingin memperdalam kemampuan dalam pengembangan aplikasi web menggunakan framework Laravel secara terstruktur dan praktis. Melalui bootcamp ini, saya berharap dapat memahami konsep dasar hingga lanjutan Laravel, menerapkannya dalam pembuatan proyek nyata, serta meningkatkan keterampilan problem solving dalam pemrograman. Harapan saya setelah mengikuti bootcamp ini adalah mampu membangun aplikasi web yang berkualitas, efisien, dan sesuai dengan kebutuhan industri, sehingga dapat menjadi bekal berharga untuk mengembangkan karier di bidang web development.
